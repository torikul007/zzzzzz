# Installation Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/torikul007/zzzzzz.git
   ```
2. Navigate into the project directory:
   ```bash
   cd zzzzzz
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```
